# Trauma-Informed Language Principles

## Core Principles
1. **Safety** - Language that creates psychological safety
2. **Trustworthiness** - Transparent, consistent messaging
3. **Choice** - Language that offers agency
4. **Collaboration** - Inclusive, empowering language
5. **Empowerment** - Strength-based framing

## Language Guidelines

### ✅ DO Use:
- Person-first language ("person experiencing homelessness")
- Neutral descriptors
- Empowering verbs ("access support" not "receive help")
- Clear, plain language

### ❌ AVOID:
- Stigmatizing labels ("addict," "victim")
- Blame-oriented language
- Triggering descriptors
- Assumptions about experiences
